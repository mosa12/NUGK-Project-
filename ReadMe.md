# PollVault - Modern Polling Application

A real-time polling application with a modern, engaging UI where users can create polls, vote, and see instant visual results.

## Features

- 🗳️ Create polls with 2-4 options
- 📊 Real-time result updates with animated progress bars
- 🎨 Modern gradient UI with smooth animations
- 🚫 Duplicate vote prevention (IP-based + client-side)
- 📱 Fully responsive design
- 🎯 Percentage calculations with vote counts
- ✨ Toast notifications for user feedback

## Tech Stack

- **Backend**: Node.js, Express
- **Frontend**: Vanilla HTML5, CSS3, JavaScript
- **Styling**: CSS Custom Properties, Gradients, Animations
- **Icons**: Font Awesome
